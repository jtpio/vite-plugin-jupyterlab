# create-jupyterlab-extension

Scaffold a JupyterLab extension powered by Vite and `vite-plugin-jupyterlab`.

## Usage

```bash
pnpm create jupyterlab-extension my-extension
```

You can also run the package directly:

```bash
npx create-jupyterlab-extension my-extension
```

The CLI prompts for one of four extension kinds:

- Frontend
- MIME renderer
- Theme
- Frontend + server

Generated projects include JavaScript, Python packaging metadata, and the
JupyterLab discovery files needed for local development.
